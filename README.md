# ROBOTIC ARM
Control of a robotic arm with 4 degrees of freedom.

### - MATERIALS
  * ARDUINO UNO..........x1
  * SERVO SG90...........x4
  * 10K POTENTIOMETER....x4
  * 2 AXES JOYSTICK......x2
  * PROTOBOARD...........x1
  
### - CONTROL MODES
  - With 4 potentiometers, one servo for each potentiometer.
  - With 2 joysticks, 2 servos for each joystick.
